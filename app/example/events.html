data 属性

你可以仅仅通过 data 属性就能使用的 Hound 事件处理功能，无需写一行 JavaScript 代码。

话又说回来，在某些情况下可能需要将此功能关闭。因此，我们还提供了关闭 data 属性 API 的方法，即解除以 data-api 为命名空间并绑定在文档上的事件。就像下面这样：

<button data-click="redirect" data-url="/index.html">返回首页</button>
